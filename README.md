Titanic-Classification-ML

Titanic Survival Prediction

About this Dataset: The Titanic dataset is one of the most popular beginner-friendly datasets in machine learning. It contains information about passengers aboard the RMS Titanic, such as age, gender, ticket class, fare, and whether they survived or not. This dataset is widely used to practice classification, data preprocessing, and feature engineering techniques.

This project aims to build a machine learning model to predict passenger survival on the Titanic using various features and data analysis techniques. The dataset includes important attributes such as - Pclass, Sex, Age, Fare, Embarked, and family-related features.

In this project, I performed:

* Data cleaning and handling missing values (`Age`, `Embarked`, `Cabin`)
* Feature engineering (e.g., `AgeGroup`, `FamilySize`)
* Encoding categorical variables using get_dummies
* Exploratory Data Analysis (EDA) with visualizations
* Model building using Logistic Regression and another ML models

The model achieved **87.43% accuracy**, demonstrating strong performance on the dataset.
